## 🧾 주요기능

#### 핵심기능

- 사용자의 얼굴을 눈, 코, 입, 눈썹으로 분할하여 세부적인 관상 정보를 제공합니다.
- 관상 정보를 종합하여 조선시대 직업으로 나타냅니다.
- 이름 두개를 입력받아 궁합을 계산합니다.
- 사용자의 나이를 입력받아 하루 24시간으로 변환합니다.
